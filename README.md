## 🚀 Proyectos destacados

### 🚛 [flota-saas](https://github.com/blues182/flota-saas)
Sistema SaaS para administrar flotillas de hasta 20 vehículos: inventario de unidades, control de cambios de aceite, vencimiento de licencias, mantenimientos futuros y balance general. Diseñé la base de datos en PostgreSQL y desarrollé la lógica de negocio en TypeScript.

### ♟️ [juego-estrategia](https://github.com/blues182/juego-estrategia)
Juego de estrategia web por ticks inspirado en juegos de gran estrategia histórica.

### 🏛️ Sistema de reservas de salones
Aplicación desarrollada en equipo con Python (Tkinter) y MariaDB para gestionar la reserva de salones de una institución educativa, con despliegue en Docker.

### 🅿️ Sistema de estacionamiento con FPGA
Diseño de la lógica de control en Verilog, implementado sobre una FPGA.
